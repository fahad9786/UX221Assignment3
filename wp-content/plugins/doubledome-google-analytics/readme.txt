**How to Enable Google Analytics 4:**

1. Create a GA4 account [Click here](https://support.google.com/analytics/answer/9306384)
2. Note down your tracking ID.
3. Enter your tracking ID in the plugin settings.
4. Choose where the tracking code will be placed (not following Google's default recommendation).
5. Optionally, configure any custom GTag objects.
6. Add any custom code you want to include with your tracking script.
7. Set the location for your custom code if needed.

**Features:**

1. Easy GA4 Tracking ID entry
2. Flexible script location settings
3. Option to add custom GTag objects
4. Ability to include custom code and specify its location
5. Email support for the plugin


== How To Install DoubleDome Google Analytics plugin ==

This very simple plugin allows you to quickly connect your website to your Google Analytics account for complete tracking. Within one setting view you'll be able to add your GA 4 ID, page location, and any desired custom GTag objects and custom code.


== Installation ==

1. Upload the entire `google-analytics-dd` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Configure Google Analytics 4 ID? =
Click on "DoubleDome GA4" in left sidebar menu of Wordpress Admin. On this page, click on "PLUGIN SETTINGS" and enter Google Analytics 4 Measurement ID into "Google Analytics 4 Measurement ID" field. Select the Google Analytics Tag location placement and click on "Save Changes" button. That's it. Google Analytics tracking will start working.
